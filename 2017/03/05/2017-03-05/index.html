写于一个无聊的周末

2017-03-05

《红楼梦》随笔系列之开篇

周日，除了拿了个快递，也就没有出门。这么宅还不够，为了图清净，把电脑拿到诺大空旷的会议室，一个人待了大半日，看论文，发呆，逛淘宝，间或刷刷社交网络，给自己一种还处于人际关系网中的虚假的安慰。趴在会议室的圆形办公桌上，望着窗外灯火璀璨的夜景，看着时钟滴滴答答的走着，任凭思绪蔓延到空间的每个角落。

思想是有触角的，总会因为这样那样的原因拘束着，只有在这空旷无人的空间中才得以小心翼翼的舒展一下。去田野去森林空间更大更好，而公共空间却没了独自一人的自在感。

我大约真的挺宅的，不过二十多年都这么过来了，一个人倒也会自得其乐。也就近些日子萌发了想要多接触人群的意愿，慢慢开始懂得思考些人情事故，思考与人相处的艺术。虽然不想承认这可能是年龄大了的关系，但心智也确乎需要汲取新鲜的养分了，艺术根植于泥土，思想又何尝不是呢。

近来为了少胡思乱想，一直不停的塞给自己事情忙，而且是不需要多少智商的事情，脑子专注于做事情，反而有一种久违的轻松感。脑袋放空了，就变得清明起来。影视作品一概不看，书也懒得读，只是没事略翻一翻《红楼梦》，不紧不慢地，反而发现了很多有意思的细节，也得了许多感悟，一个个人物越发清晰，读着文字，倒像真的亲历了一般。

很想把这些思考写下来。对笔下这些可爱可敬的人物理解的越多，越是能体会到曹公的一片痴心。无奈自小文笔便不好，总是不能找到合适的表达方式。写这些文章的同时，也是自我探寻，建立自己与世界沟通的方式的一种探索吧。务必以坚持为紧要。是以为记。
